Automated Server Backup Script

While transitioning into a new role within my organization I identified an opportunity to automate the daily backup of files from a UNIX server which was being completed manually. Utilizing WinSCP, SFTP, & Windows Task Scheduler, I was able to write a batch script to pull files from a server and compress and archive these files to a local network drive at a prescribed time while also performing some data validation checks on the files. The process improvement helped to significantly reduce inherent risk in the process related to human error and will result in an estimated 65 hours of time savings per year (15mins/day X 261 working days).
