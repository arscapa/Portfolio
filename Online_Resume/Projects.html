Automated Server Backup Script

Recently identified an opportunity to automate the daily backup of files from a UNIX server which was being completed manually within my organization. I was able to write a batch script to pull files from the server, compress and archive these files at a scheduled time, while also performing some data validation checks on the files. This improvement significantly reduced inherent risk in the process and will result in an estimated 65 hours of time savings per year (15mins/day X 261 working days). Implemented utilizing WinSCP, SFTP, 7-Zip, Windows Task Scheduler, Batch Scripts, and Python coding.
